# MathExam

Übungsaufgabe

---

```
Erstell ein Programm für eine Matheprüfung.

Das Programm soll folgende Anforderungen erfüllen:
- Das Programm zeigt am Anfang nur den Button "Test starten!"
- Beim Klick auf den Button werden fünf Additionsaufgaben (z.B. 12 + 34) mit zufällig generierten Zahlen angezeigt
- Zu jeder Additionsaufgabe gehört ein Eingabefeld, in welches das Ergebnis eingetragen werden kann
- Irgendwo wird ein Countdown angezeigt, der dem Prüfling 30 Sekunden Zeit gibt, die richtigen Aufgaben einzutragen
- Ein Button "Fertig" prüft die Richtigkeit der Ergebnisse und ist nur während der laufenden Prüfung anklickbar

Tipp:
var rng = new Random()
var randomNumber = rng.Next(1,100);
// Erzeugt eine Zufallszahl zwischen 1 und 100
```
