# MovingSquares

Übungsaufgabe

---

```
In dieser Aufgabe bewegen sich mehrere farbige Quadrate.

Die Bewegung der Quadrate wird wird über 4 Methoden gesteuert: MoveUp, MoveDown, MoveLeft und MoveRight.
Hier bewegt sich allerdings noch gar nichts. Implementiere die Methoden in der Datei Form1.cs.

Das Programm hat weiterhin mehrere Knöpfe: Start, Stop, Schneller, Langsamer.
Die Knöpfe haben bisher aber noch keine Funktion. Implementiere die Funktionen der Knöpfe, wie in den jeweiligen Methoden in der Datei Form1.cs beschrieben.
```
